# Public Directory

This directory contains static assets served by Next.js.

## Required Files

- `headshot.jpg` - Your professional headshot image (recommended: 800x800px or larger, square format)

## Optional Files

- `projects/` - Directory for project images
  - Place project screenshots/images here
  - Update image paths in `lib/data.ts` to reference these files

## Image Guidelines

- Use optimized images (JPG, PNG, or WebP format)
- Recommended sizes:
  - Headshot: 800x800px (square)
  - Project images: 1200x600px or larger (16:9 or 4:3 ratio)
- Optimize images before adding them to reduce file size
